# Blockchain workshop

Install:
```
yarn
```

Build:

```
yarn build

// Contracts only
yarn build:contracts

// TypeScript only
yarn build:ts
```

Start `src/HeadTail.ts`:
```
yarn start
```

Test:
```
yarn test
```

Make sure to start local blockchain first:

```
yarn start:ganache
```
